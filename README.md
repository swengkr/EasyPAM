## 개요
소규모 인프라 관리를 위한 **PAM(Privileged Access Management)** 솔루션을 개발하기 위해 요구사항 분석 중 입니다.

## 주요 기능
+ 대시보드
  - 실시간 데이터 조회
  - 사용자 데이터 시각화
  - 자산 데이터 시각화
  - 계정 / 자산 활동 상태 통계
  - 자산 유형 비율 표시
  - 로그인 계정 / 자산 순위
+ 조직도 관리
  - 부서 조회 / 추가 / 수정 / 삭제
+ 사용자 관리
  - 사용자 조회 / 추가 / 수정 / 삭제
  - 사용자 그룹 조회 / 추가 / 수정 / 삭제
+ 자산 관리
  - 자산 조회 / 추가 / 수정 / 삭제
  - 자산 계정 매핑
+ 계정 관리
  - 계정 조회 / 추가 / 수정 / 삭제
  - 콘솔 로그인 / 로그아웃
+ 권한 관리
+ 접근 관리
  - 접속 이력 조회 / 신청 / 수정
  - 자산 원격 접속 승인
  - 자산 원격 접속
  - 자산 접속 모니터링 / 제어
+ 공지사항
  - 공지사항 조회 / 추가 / 삭제
+ 자료실
  - 자료실 조회 / 추가 / 수정 / 삭제
+ 환경관리
  - 시스템 설정
